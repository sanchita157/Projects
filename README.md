# Superstore Sales Dashboard

## Project Overview
This project is an interactive Excel Dashboard created using Superstore Sales data.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers

## Dashboard Features
- Total Sales
- Total Profit
- Total Quantity
- Total Orders
- Sales by Category
- Sales by Segment
- Sales by Market
- Top 10 Customers
- Interactive Slicers

## Skills Demonstrated
- Data Cleaning
- Pivot Tables
- Pivot Charts
- Dashboard Design
- Data Visualization
